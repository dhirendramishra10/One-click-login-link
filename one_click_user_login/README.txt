
Description
-----------
This module provide option to user to Generate One click Login Link on your Drupal site.
User can keep use of this feature or switch off(if they don't want to use) using on/off flag provided in "generate login link"  tab under "/user" link

Requirements
------------
Drupal 7.x


Installation
------------
1. Copy the One click Login Link to the Drupal sites/all/modules directory.

2. Login as an administrator. Enable the module in the "Administer" -> "Modules"

3. Give the permission to role for Generate One Click Login Link  under "Administer" -> "permission" section

4. Login with permitted role user and goto "/user" in URL and click on "Generate Login Link".
